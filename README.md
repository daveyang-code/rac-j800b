# RAC-J800B USB-C Mod

This is a simple mod to replace the USB-B jack on the RAC-J800B with a much more reliable USB-C jack.

## Required Materials

### USB-C Bracket

USB-C Breakout Board https://www.adafruit.com/product/5180

4-PIN JST-PH Connector

### SOCD Switch

Seeed Xiao SAMD21

Slide switch

M2x6 (2), M3x4 (2)
